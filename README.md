**Straw return enhances soil multifunctionality by promoting protist-dominated microbial multitrophic interactions**

This repository contains the  data and code for the paper "Straw return enhances soil multifunctionality by promoting protist-dominated microbial multitrophic interactions" published in Soil & Tillage Research.

All data files are organized in the data/ directory, containing soil properties and ASVs tables. Raw sequencing data (FASTQ files) can be find through: https://nmdc.cn/resource/genomics/project/detail/NMDC10019327 (Bacteria and Fungi) and https://nmdc.cn/resource/genomics/project/detail/NMDC10019901 (Protist).

Operating system: Windows/Linux
R version: 4.3.3
R packages: tidyverse, igraph, vegan, Hmisc, ggpubr, etc.
